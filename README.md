# 个人博客
练手造的一个个人博客轮子，
目前还在开发完善，持续更新中。。。

## 技术栈
前端: **Vue + Vuex + VueRouter + view**

后端: **node + mongdb + passport**

[后端代码](https://github.com/nativejie/vue-blog-server)

## 已实现功能
+ **文章添加**
+ **文章修改**
+ **文章删除**
+ **标签增加**
+ **类型增加**
+ **列表和首页**


## 运行
```node
1. npm run install
2. npm run serve
```
## 打包
```node
npm run build
```
